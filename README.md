# BucketDimension

Bucket Dimension plugin (Maven).

## Build
- Install Java 21+
- Install Maven 3.9+

```bash
mvn -q package
```

Jar output:
- `target/bucket-dimension-1.0.0.jar`

## Notes
- Paper API version is currently set to `1.21.1-R0.1-SNAPSHOT` in `pom.xml`.
- If you're on a different Paper version, edit `<paper.api.version>...</paper.api.version>`.
